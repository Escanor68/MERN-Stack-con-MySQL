# MERN-Stack-con-MySQL

## Proyecto
Creando una aplicaci√≥n web fullstack usando el Stack MERN (MySQL, Express, React, Node), solo que usaremos en nuestro caso MySQL.     3- Correr el script npm run start para levantar el proyecto y en funionamiento
